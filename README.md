🚀 Installation & Running Locally
📥 Clone the repository:

```bash
git clone https://github.com/alibelhrak/Portfolio
```
📌 Navigate to the project directory:

```bash
cd app.js
```
📦 Install dependencies:
```bash
npm install
```
▶️ Run the app locally:
```bash
ng serve
```
Then open http://localhost:4200 🚀

